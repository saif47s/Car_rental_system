# 🚗 HK Gari Rental Manager

A complete, 100% offline standalone Windows Desktop Application built with **Electron.js** and **HTML/JS** for managing a car rental business.

## 🌟 Overview
This system allows rental business owners to track their vehicles, customers, daily expenses, and overall profit/loss without needing an active internet connection or any backend server like XAMPP. All data is stored securely offline on the user's local machine.

## 🚀 Quick Start (Development)
If you want to run or build the project yourself:
1. Install [Node.js](https://nodejs.org/)
2. Clone this repository
3. Run `npm install` to install dependencies
4. Run `npm start` to run the app in development mode
5. Run `npm run build` to generate the Windows executable (`.exe`)

## 📥 Download
To just use the app, download the `.exe` file from the **Releases** section (if uploaded) and install it on your Windows PC.
